dsfr-data-a11y

dsfr-data-a11y est le companion d'accessibilite unifie pour les visualisations. Il offre trois alternatives accessibles aux graphiques : tableau de donnees, telechargement CSV et description textuelle.

Pourquoi ce composant ?

Les graphiques (canvas, SVG) sont difficilement interpretables par les lecteurs d'ecran. dsfr-data-a11y resout ce probleme en offrant :

  1. Un tableau HTML accessible avec les donnees du graphique
  2. Un bouton de telechargement CSV des donnees brutes
  3. Une description textuelle libre du contenu du graphique

Via l'attribut for, il injecte automatiquement un skip link dans le graphique cible (visible au focus clavier), pose aria-describedby vers un resume concis et aria-details vers le tableau de donnees.

Usage basique

Placez dsfr-data-a11y juste apres votre composant de visualisation. L'attribut for cree automatiquement la liaison ARIA et le skip link.

<dsfr-data-source id="src" api-type="opendatasoft"
  base-url="https://data.economie.gouv.fr"
  dataset-id="mon-dataset">
</dsfr-data-source>
<dsfr-data-query id="data" source="src"
  group-by="region" aggregate="population:sum:total">
</dsfr-data-query>

<dsfr-data-chart id="mon-graph" source="data" type="bar"
  label-field="region" value-field="total"
  databox
  databox-title="Population par region"
  databox-source="data.economie.gouv.fr"
  databox-download>
</dsfr-data-chart>
<!-- Avec DataBox : description seulement (table et CSV fournis par DataBox) -->
<dsfr-data-a11y for="mon-graph" source="data"
  description="L'Ile-de-France domine largement.">
</dsfr-data-a11y>

Lignes de référence

L'attribut reference-lines superpose des repères sur les graphiques cartésiens (line, bar, bar-line, scatter) : ligne verticale à une catégorie/date (axis:"x") ou horizontale à un seuil (axis:"y"), avec un libellé en pastille. Les repères sont aussi annoncés dans l'aria-label du graphique pour les lecteurs d'écran.

<dsfr-data-chart id="mon-graph" source="data" type="line"
  label-field="mois" value-field="ventes"
  reference-lines='[
    {"axis":"x","value":"2026-02","label":"Lancement du plan","color":"#c9191e"},
    {"axis":"y","value":30000,"label":"Objectif","position":"start"}
  ]'>
</dsfr-data-chart>

Avec description textuelle

Ajoutez description pour fournir une transcription du graphique :

<dsfr-data-a11y for="mon-graph" source="data"
  table download
  description="Ce graphique montre la repartition par region.">
</dsfr-data-a11y>

Mode manuel (no-auto-aria)

Si vous gerez les attributs ARIA manuellement, ajoutez no-auto-aria pour desactiver l'injection automatique du skip link et des attributs ARIA : 

<dsfr-data-a11y source="data"
  no-auto-aria table download>
</dsfr-data-a11y>

Attributs

AttributTypeDefautDescription
sourceString""ID du composant source de donnees
forString""ID de l'element cible pour skip link + ARIA
tableBooleanfalseActive le tableau de donnees
downloadBooleanfalseActive le bouton CSV
filenameString"donnees.csv"Nom du fichier telecharge
descriptionString""Description textuelle du graphique
label-fieldString""Colonne pour les labels du tableau
value-fieldString""Colonne(s) pour les valeurs (separees par virgules)
labelString""Libelle personnalise de la section accessible
no-auto-ariaBooleanfalseDesactive ARIA automatique et skip link

Si ni table, ni download, ni description ne sont definis, les trois sont actives par defaut.
