Dashboard Commerce - API Huwise

Tableau de bord des exportations francaises 2023 avec carte mondiale interactive. Utilise dsfr-data-world-map pour visualiser les donnees geographiques et dsfr-data-source avec agregation server-side OpenDataSoft.

Composant déprécié

Depuis DSFR Chart 2.1, préférez <dsfr-data-chart type="map-monde"> (carte mondiale native, sans bundle supplémentaire). dsfr-data-world-map sera retiré dans une prochaine version majeure — cette page reste utile si le zoom continent interactif est requis.

dsfr-data-world-map : Ce composant affiche une carte choropleth du monde basee sur les donnees d'une source. Il supporte les codes ISO alpha-2, alpha-3 et numeriques via l'attribut code-format, le zoom par continent, les palettes DSFR et les tooltips avec unite personnalisee.

Architecture du pipeline

dsfr-data-source "src-pays" (ODS, group-by pays) --> dsfr-data-query "top15" (limit 15) --> dsfr-data-chart (bar) --> dsfr-data-query "top8" (limit 8) --> dsfr-data-chart (pie) --> dsfr-data-world-map (carte mondiale) dsfr-data-source "src-global" (ODS, somme globale) --> dsfr-data-kpi x3 (valeur, flux, masse) dsfr-data-source "src-pays" --> dsfr-data-kpi (count pays)

Exportations francaises par pays (carte choropleth)

Source : Statistiques nationales du commerce exterieur — data.economie.gouv.fr
API : OpenDataSoft avec agregation server-side (group-by, sum, count)
Filtre : column_1 = 'E' (exportations) AND column_3 = 2023 (annee)
Carte : dsfr-data-world-map avec codes ISO alpha-2, palette sequentielle et zoom par continent 

<!-- Source : agregation serveur par pays (exports 2023) -->
<dsfr-data-source id="src-pays"
  api-type="opendatasoft"
  base-url="https://data.economie.gouv.fr"
  dataset-id="statistiques-nationales-du-commerce-exterieur"
  select="column_7 as pays, sum(column_8)/1000 as valeur,
          count(*) as nb_flux, sum(column_9)/1000 as masse"
  where="column_1 = 'E' AND column_3 = 2023"
  group-by="column_7"
  order-by="valeur desc">
</dsfr-data-source>

<!-- Source pour KPI globaux -->
<dsfr-data-source id="src-global"
  api-type="opendatasoft"
  base-url="https://data.economie.gouv.fr"
  dataset-id="statistiques-nationales-du-commerce-exterieur"
  select="sum(column_8)/1000000 as valeur_mds,
          count(*)/1000000 as nb_flux_m,
          sum(column_9)/1000000000 as masse_mt"
  where="column_1 = 'E' AND column_3 = 2023">
</dsfr-data-source>

<!-- KPIs -->
<dsfr-data-kpi source="src-global" value="valeur_mds"
  label="Valeur exportee (Mds)" format="decimal"
  icon="ri-money-euro-circle-line" color-token="bleu">
</dsfr-data-kpi>

<dsfr-data-kpi source="src-pays" value="pays:count"
  label="Pays partenaires" format="nombre"
  icon="ri-global-line" color-token="vert">
</dsfr-data-kpi>

<!-- Top 15 : bar chart -->
<dsfr-data-query id="top15" source="src-pays"
  order-by="valeur:desc" limit="15">
</dsfr-data-query>
<dsfr-data-chart source="top15" type="bar"
  label-field="pays" value-field="valeur"
  horizontal selected-palette="sequentialDescending"
  unit-tooltip="M">
</dsfr-data-chart>

<!-- Top 8 : pie chart -->
<dsfr-data-query id="top8" source="src-pays"
  order-by="valeur:desc" limit="8">
</dsfr-data-query>
<dsfr-data-chart source="top8" type="pie"
  label-field="pays" value-field="valeur"
  selected-palette="categorical"
  unit-tooltip="M">
</dsfr-data-chart>

<!-- Carte mondiale -->
<dsfr-data-world-map source="src-pays"
  code-field="pays"
  value-field="valeur"
  code-format="iso-a2"
  name="Exportations francaises"
  unit-tooltip="M"
  selected-palette="sequentialAscending"
  zoom-mode="continent">
</dsfr-data-world-map>

Points cles de cet exemple

  • dsfr-data-world-map : carte choropleth mondiale. Attributs principaux : code-field (champ contenant le code pays), value-field (champ numerique), code-format (iso-a2, iso-a3 ou iso-num), zoom-mode="continent" (boutons de zoom par continent).
  • Agregation server-side : dsfr-data-source utilise select avec sum() et count(), group-by et order-by pour agreger directement cote serveur OpenDataSoft.
  • Pipeline de transformation : dsfr-data-query applique limit et order-by sur les donnees agregees pour alimenter les graphiques top 15 et top 8, tandis que la carte recoit toutes les donnees.
  • KPI derives : value="pays:count" compte le nombre de valeurs du champ pays sans source supplementaire.
  • Accessibilite : dsfr-data-a11y sur chaque graphique pour le telechargement CSV.
  • 0 ligne de JavaScript : tout le dashboard est 100% declaratif en HTML.
