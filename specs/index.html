dsfr-data est une bibliotheque de Web Components pour creer des tableaux de bord et visualisations de données conformes au Design System de l'État (DSFR).

Architecture

Cette bibliotheque s'articule autour de deux types de composants complementaires :

Composants dsfr-data

Composants developpes specifiquement pour ce projet. Ils gerent la connexion aux données, les transformations, et l'affichage d'indicateurs. Ils communiquent entre eux via un systeme de data-bridge qui permet de partager les données sans ecrire de JavaScript.

Graphiques DSFR Chart

Graphiques provenant de la bibliotheque officielle DSFR Chart. Ces composants sont conformes RGAA et respectent la charte graphique de l'État. Ils peuvent etre utilises directement ou via notre wrapper dsfr-data-chart pour une alimentation dynamique.

Le pont entre les deux : Le composant <dsfr-data-chart> fait le lien entre le systeme de données dsfr-data-source et les graphiques DSFR Chart, permettant de creer des visualisations dynamiques alimentees par une API.

Composants dsfr-data

Ces composants gèrent la recuperation, la transformation et l'affichage des données.

dsfr-data-source

Connecteur de données

Recupere les données depuis une API et les distribue aux autres composants.

dsfr-data-normalize

Normalisation des données

Nettoie et normalise les données : conversion numérique, renommage, trim.

dsfr-data-query

Requetage des données

Filtre, trie et agrège les données issues d'une source.

dsfr-data-join

Jointure multi-sources

Joint deux sources de données sur une clé pivot pour enrichir un jeu de données.

dsfr-data-unpivot

Bascule "wide" vers "tidy"

Deplie un tableur ou le temps est dans les noms de colonnes (c2023_01...) en une observation par ligne.

dsfr-data-facets

Filtres a facettes

Ajoute des filtres interactifs par facettes sur les données d'une source.

dsfr-data-search

Recherche textuelle

Champ de recherche DSFR qui filtre les données en amont. Se combine avec les facettes.

dsfr-data-list

Tableau de données

Tableau avec recherche, filtres, tri, pagination et export CSV.

dsfr-data-display

Affichage dynamique

Genere des cartes, tuiles ou tout motif HTML repetitif a partir d'un template et de données.

dsfr-data-kpi

Indicateur chiffre

Affiche une valeur clé avec formatage, couleurs conditionnelles et icones.

dsfr-data-podium

Podium et classement

Affiche un classement sous forme de podium avec medailles, couleurs et icones personnalisables.

dsfr-data-chart

Wrapper graphique

Connecte les graphiques DSFR Chart au systeme de données dynamiques.

dsfr-data-world-map

Carte du monde

Carte choropleth avec zoom interactif par continent et tooltips.

dsfr-data-map

Carte interactive

Carte Leaflet multi-couches avec POI, geoshape, cercles, clustering et chargement par viewport.

dsfr-data-a11y

Accessibilité graphique

Companion d'accessibilité unifie : tableau des données, téléchargement CSV et description textuelle.

Graphiques DSFR Chart

Graphiques officiels conformes au DSFR. Utilisez-les directement ou via dsfr-data-chart pour des données dynamiques.

line-chart

Courbes et tendances

Visualiser des evolutions dans le temps.

bar-chart

Barres et comparaisons

Comparer des valeurs entre catégories.

pie-chart

Camembert et proportions

Representer des distributions et parts.

radar-chart

Radar multidimensionnel

Comparer plusieurs variables simultanement.

gauge-chart

Jauge de progression

Afficher une progression vers un objectif.

map-chart

Carte de France

Visualiser des données geographiques.

scatter-chart

Nuage de points

Analyser des correlations entre variables.
