Filtrer, regrouper ou transformer les données avant affichage

Le mode avance utilise <dsfr-data-query> pour filtrer, regrouper et agréger les données cote client avant de les afficher.

Operateurs disponibles
  • eq = egal
  • neq = different
  • gt / gte = superieur (strict / ou egal)
  • lt / lte = inferieur (strict / ou egal)
  • contains = contient (texte)
  • in = dans la liste (val1|val2|val3)
  • isnull / isnotnull = null ou non
